# Kernel Signal State Ownership (sub-project B) — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make the kernel the sole authority for signal *state* (per-thread blocked mask, two-pool pending, per-process disposition) and the deliver/defer *decision*, with thin kernel-routed guest shims — Stage 1 of the signal-delivery redesign that re-lands #90's mask correctly and unblocks #91.

**Architecture:** New `PendingSet` newtype (process-directed + per-thread); `ThreadRecord.blocked_signals`; `SigDisposition` replaces opaque `[u32;63]`; one shared `pend()` funnel (SIG_IGN-drop, SIGKILL/SIGSTOP, pool-routing, job-control mutual-discard at produce-time); kernel-authored `sys_signal_raise` action verdict so the guest never adjudicates; `sys_signal_query` primitive for #91. Handler execution / `sigreturn` / drain / preemption are sub-project (C) — NOT here.

**Tech Stack:** Rust (`packages/kernel-wasm`), C ABI shims (`abi/src`, `abi/include`), TOML ABI contract (`abi/contract/yurt_abi_methods.toml`), generated `methods_generated.rs` (via `packages/kernel-wasm/build.rs`).

**Spec:** `docs/superpowers/specs/2026-05-17-kernel-signal-state-ownership-design.md` (round 9). **Umbrella:** `…/2026-05-17-kernel-signal-delivery-architecture-vision.md`. **Prior art (reuse, don't rebuild):** the #90 rounds-1–7 spec on branch `worktree-parity-signal-mask` at `docs/superpowers/specs/2026-05-17-signal-mask-surface-design.md` — the 1-byte compact `sigset_t` ⇄ `1<<(sig-1)` `expand`/`narrow` + `SLOTS` table, the `process_mut` sibling convention.

**Plan-level decisions resolving two spec under-specifications (within spec intent, not design changes):**
1. **Evolved `sys_sigwaitinfo` standard-signal siginfo wire:** keep the existing 16-byte layout `{i32 si_signo, i32 si_code, u32 si_pid, i32 si_value}` LE. RT entry: `si_code = SI_QUEUE(-1)`, `si_pid = sender_pid`, `si_value = value` (unchanged). **Standard** (bitmask) entry accepted from a pool: `si_code = 0x80` (`SI_KERNEL`), `si_pid = 0`, `si_value = 0`. One encoder, two cases.
2. **Gated `sigsuspend`/`sigtimedwait` keep `yurt_host_yield()`** before returning (today's behavior at `yurt_signal.c:299,338`) so a `while`-loop caller does not hot-spin — preserves current CPU behavior; no interim regression beyond the documented under-delivery.

```rust
use crate::kernel::{with_kernel, RtSignal};
const SIGKILL: u32 = 9; const SIGSTOP: u32 = 19;
const SIGCONT: u32 = 18;
const STOPSIGS: &[u32] = &[SIGSTOP, 20 /*SIGTSTP*/, 21 /*SIGTTIN*/, 22 /*SIGTTOU*/];
const UNMASKABLE: u64 = (1u64 << (SIGKILL-1)) | (1u64 << (SIGSTOP-1));

/// Which pool a producer targets.
pub enum Pool { Process, Thread(u32 /*tid*/) }

/// The SINGLE producer funnel. `disp_ignored`: caller passes whether the
/// per-process disposition for `sig` is SIG_IGN. Returns true if pended.
pub(crate) fn pend(p: &mut crate::kernel::Process, pool: Pool, sig: u32, rt: Option<RtSignal>, disp_ignored: bool) -> bool {
    if !(1..=63).contains(&sig) { return false; }
    if disp_ignored { return false; }                 // SIG_IGN => discard, never enqueue
    // Job-control mutual-discard at produce-time, both pools:
    if sig == SIGCONT { purge(p, STOPSIGS); }
    if STOPSIGS.contains(&sig) { purge(p, &[SIGCONT]); }
    let bit = 1u64 << (sig - 1);
    let set = match pool {
        Pool::Process => &mut p.pending,
        Pool::Thread(tid) => match p.threads.get_mut(&tid) { Some(t) => &mut t.pending, None => return false },
    };
    match rt { Some(r) => set.rt.push_back(r), None => set.standard |= bit }
    true
}
fn purge(p: &mut crate::kernel::Process, sigs: &[u32]) {
    for &s in sigs {
        let b = 1u64 << (s - 1);
        p.pending.standard &= !b; p.pending.rt.retain(|r| r.signo != s);
        for t in p.threads.values_mut() { t.pending.standard &= !b; t.pending.rt.retain(|r| r.signo != s); }
    }
}
/// disposition→IGN purge: process-wide (process pool + EVERY thread pool).
pub(crate) fn purge_ignored(p: &mut crate::kernel::Process, sig: u32) { purge(p, &[sig]); }
/// pending ∧ ¬blocked for thread `tid` (disposition handled by callers/raise).
pub(crate) fn thread_has_deliverable(p: &crate::kernel::Process, tid: u32) -> bool {
    let blk = p.threads.get(&tid).map(|t| t.blocked_signals).unwrap_or(0);
    let std_un = (p.pending.standard | p.threads.get(&tid).map(|t| t.pending.standard).unwrap_or(0)) & !blk;
    if std_un != 0 { return true; }
    let rt_un = |q: &std::collections::VecDeque<RtSignal>| q.iter().any(|r| (1..=63).contains(&r.signo) && blk & (1u64<<(r.signo-1)) == 0);
    rt_un(&p.pending.rt) || p.threads.get(&tid).map(|t| rt_un(&t.pending.rt)).unwrap_or(false)
}
```
Tests: `pend(SIG_IGN=true)` returns false + nothing enqueued; `pend(SIGCONT)` purges a pre-seeded `SIGSTOP` from process + a thread pool; `pend(SIGSTOP)` purges pending `SIGCONT`; `thread_has_deliverable` false when the only pending sig is blocked, true when unblocked.

use crate::abi;
use crate::dispatch::DispatchContext;
const SIG_BLOCK:i32=0; const SIG_UNBLOCK:i32=1; const SIG_SETMASK:i32=2;
pub(super) fn sys_sigprocmask(ctx: DispatchContext, request: &[u8], response: &mut [u8]) -> i64 {
    if request.len()!=6 || response.is_empty() { return -(abi::EINVAL as i64); }
    let how=i32::from_le_bytes(request[0..4].try_into().expect("4"));
    let has=request[4]!=0; let set=expand(request[5]);
    with_kernel(|k| {
        let p=k.process_mut(ctx.caller_pid);
        let Some(t)=p.threads.get_mut(&ctx.caller_tid) else { return -(abi::ESRCH as i64) };
        response[0]=narrow(t.blocked_signals);
        if has {
            let n=match how { SIG_BLOCK=>t.blocked_signals|set, SIG_UNBLOCK=>t.blocked_signals&!set, SIG_SETMASK=>set, _=>return -(abi::EINVAL as i64) };
            t.blocked_signals = n & !UNMASKABLE;
        }
        1
    })
}

pub(super) fn sys_signal_raise(ctx: DispatchContext, request: &[u8], response: &mut [u8]) -> i64 {
    if request.len()!=4 || response.len()<8 { return -(abi::EINVAL as i64); }
    let sig=u32::from_le_bytes(request[0..4].try_into().expect("4"));
    if !(1..=63).contains(&sig) { return -(abi::EINVAL as i64); }
    with_kernel(|k| {
        let p=k.process_mut(ctx.caller_pid);
        let d=p.signal_dispositions[(sig-1) as usize];
        let blocked = p.threads.get(&ctx.caller_tid).map(|t| t.blocked_signals & (1u64<<(sig-1)) != 0).unwrap_or(false);
        let (action, token): (i32,u32) = if blocked {
            pend(p, Pool::Thread(ctx.caller_tid), sig, None, false); (0,0) // NONE, thread-pended
        } else if d.handler==1 { (0,0) } // SIG_IGN => NONE discard
        else if d.handler!=0 { (1, d.handler) } // RUN_HANDLER + token
        else { // SIG_DFL => kernel resolves default-action class
            match default_action(sig) { Dfl::Term=>(2,0), Dfl::Stop=>(3,0), Dfl::Cont=>(4,0), Dfl::Ign=>(0,0) }
        };
        response[0..4].copy_from_slice(&action.to_le_bytes());
        response[4..8].copy_from_slice(&token.to_le_bytes());
        8
    })
}
enum Dfl { Term, Stop, Cont, Ign }
fn default_action(sig: u32) -> Dfl {
    match sig {
        17|23|28 => Dfl::Ign,            // SIGCHLD, SIGURG, SIGWINCH default-ignore
        18 => Dfl::Cont,                  // SIGCONT
        19|20|21|22 => Dfl::Stop,         // SIGSTOP/TSTP/TTIN/TTOU
        _ => Dfl::Term,
    }
}
pub(super) fn sys_signal_query(ctx: DispatchContext, _r: &[u8], response: &mut [u8]) -> i64 {
    if response.is_empty() { return -(abi::EINVAL as i64); }
    let d=with_kernel(|k| { let p=k.process_mut(ctx.caller_pid); thread_has_deliverable(p, ctx.caller_tid) });
    response[0]=if d {1} else {0}; 1
}
